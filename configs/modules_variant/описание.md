## Команды проекта
- livepreload server directory `build`
```
 gulp.task('browser-sync') 
```
- create the tunnel for you site,directory `prodaction`
```
 gulp.task('server')
```
- validation and linters the files
```
 gulp.task('validation')
```
- run tests on jasmine \ mocha
```
 gulp.task('test'
```
- run in development       
```
 gulp.task('default')
```
- prodaction
```
 gulp.task('build')
```
